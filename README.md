# python-utils

Utilities written to aid in bulk photograph processing in Media Center workflow. Primarily geared towards ingest to the [Media Center Image Database](https://mcid.mcah.columbia.edu/) (MCID), with an emphasis on digitized MCAH legacy collections. Each directory has (or will have) its own documentation detailing usage and workflow. All scripts are coded to run on mac.

Unless otherwise specified, all scripts written by Lisa Schloegel-Peck on behalf of the Media Center for Art History, Columbia University, Department of Art History & Archaeology. Copyright The Trustees of Columbia University.
